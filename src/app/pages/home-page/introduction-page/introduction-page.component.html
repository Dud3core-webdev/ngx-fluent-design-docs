Getting started

To get started with this library, run either:


yarn add ngx-fluent-design

or

npm i ngx-fluent-design --save

to install the package locally.

Setup

You can build your own theme by heading to Microsoft's Fluent Design Theme Builder

Import BrowserAnimationsModule

Make sure you have imported the BrowserAnimationsModule in your app module
